# Movimentação de Peças de Xadrez

Projeto do Tema 4: demonstra a movimentação básica das peças de xadrez.

## Como usar
1. Escolha uma peça no menu.
2. Clique em uma casa do tabuleiro para definir a posição inicial.
3. As casas verdes indicam movimentos possíveis.

## Observações
- O projeto foca nos movimentos das peças.
- Não considera bloqueio por outras peças (exceto que torre/bispo/rainha vão até a borda).
